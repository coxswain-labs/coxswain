Coxswain has been deployed!

Verify the controller is ready:

  kubectl -n {{ .Values.namespace.name }} get pods -l app.kubernetes.io/name={{ include "coxswain.name" . }}
  kubectl get gatewayclass {{ .Values.gatewayClass.name }}

Check readiness and subsystem detail from inside the cluster:

  kubectl -n {{ .Values.namespace.name }} port-forward svc/{{ include "coxswain.fullname" . }}-internal {{ .Values.health.port }}:{{ .Values.health.port }} {{ .Values.admin.port }}:{{ .Values.admin.port }}
  curl http://localhost:{{ .Values.health.port }}/readyz
  curl http://localhost:{{ .Values.admin.port }}/status

{{- end }}
